# Sales Performance Analysis Dashboard

## Project Overview
Interactive Power BI dashboard analyzing sales performance data from 2015 to 2018 across 3 product categories: Technology, Furniture, and Office Supplies.

## Key Metrics
- Total Sales: $2.30M
- Total Profit: $286.40K
- Profit Margin: 12.47%
- Total Customers: 793
- Total Products: 2K+

## Dashboard Pages
- **Executive Overview** — Sales trends, total KPIs, and performance summary
- **Customer Analysis** — Top 10 customers by sales and profit, average sales per customer
- **Product Analysis** — Top products by revenue, category breakdown, sub-category performance

## Tools Used
- Power BI
- Microsoft Excel
- Power Query
- DAX
